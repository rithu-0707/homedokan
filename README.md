# HomeDokan - Online Grocery Shopping Website

## Project Overview

**HomeDokan** is a responsive online grocery shopping website developed using **HTML, CSS, JavaScript, and Bootstrap 5**. The website enables users to browse grocery products, search items, manage their shopping cart, and explore different product categories through a simple and user-friendly interface. The project focuses on creating a modern e-commerce experience for purchasing daily essentials online. 

---

## Features

* Responsive design for desktop, tablet, and mobile devices
* Attractive homepage with featured products
* Product categories

  * Vegetables
  * Fruits
  * Dairy Products
  * Cooking Essentials
  * Cleaning Essentials
  * Bakery
* Product search functionality
* Shopping cart with:

  * Add to Cart
  * Remove Items
  * Quantity Update
  * Total Price Calculation
* Login and Signup pages
* Offers page
* About Us page
* Contact Us page
* Reusable Navbar and Footer
* Dynamic product card generation using JavaScript  

---

## Cart Functionality

The shopping cart provides:

* Add products to cart
* Remove products from cart
* Update product quantity
* Automatic total price calculation
* Cart data stored using browser Local Storage
* Persistent cart even after refreshing the page 

---

## Search Functionality

The search feature allows users to:

* Search products by name
* View matching products instantly
* Navigate directly to the corresponding product page

## Responsive Design

The website is fully responsive and optimized for:

* Desktop
* Tablet
* Mobile devices

Bootstrap 5 and custom CSS media queries are used to ensure a smooth user experience across different screen sizes.
